# 🔍 OpenAI Researcher Agent

A multi-agent AI research application built using OpenAI Agents SDK and Streamlit. This project allows users to perform automated research on any topic and generates a structured report.

---

## 🚀 Features

### 🤖 Multi-Agent Architecture
- **Triage Agent** → Plans the research workflow  
- **Research Agent** → Collects data from the web  
- **Editor Agent** → Generates a structured report  

### 📊 Other Features
- Automatic fact collection with sources  
- Structured report generation (title, headings, references)  
- Interactive UI using Streamlit  
- Real-time research process tracking  

---

## 🛠️ Tech Stack

- Python 🐍  
- OpenAI Agents SDK  
- Streamlit
